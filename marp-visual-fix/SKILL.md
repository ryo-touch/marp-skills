# Marp Visual Fix スキル

Marpスライドの表示崩れやレイアウト問題を診断・修正するスキル。

## 使用タイミング

- 「スライドの見た目を修正」
- 「文字がはみ出す」
- 「画像が切れている」
- 「レイアウトがおかしい」
- 「PDFにすると崩れる」

## ワークフロー

1. **問題確認**: 現在のスライドをPNG出力して視覚的に確認
2. **原因診断**: Markdownとテーマを分析
3. **修正案作成**: 複数の修正オプションを提示
4. **修正適用**: 選択された方法で修正
5. **確認**: 修正後の画像を比較

## 参照スキル

- **marp-knowledge**: ディレクティブ、画像構文の仕様

## スクリプト

- [export_png.sh](scripts/export_png.sh) - PNG画像へエクスポート
- [compare_slides.py](scripts/compare_slides.py) - 修正前後の画像比較

## リファレンス

- [common-issues.md](references/common-issues.md) - よくある表示問題
- [fix-patterns.md](references/fix-patterns.md) - 問題別修正パターン
- [css-adjustment.md](references/css-adjustment.md) - CSS調整テクニック

## 問題診断チェックリスト

1. **テキストはみ出し**
   - フォントサイズが大きすぎないか
   - 箇条書きが多すぎないか
   - 見出しが長すぎないか

2. **画像問題**
   - サイズ指定が正しいか
   - パスが正しいか
   - アスペクト比が崩れていないか

3. **レイアウト問題**
   - ディレクティブが正しいか
   - クラス指定が正しいか
   - スライド分割位置が適切か

4. **PDFエクスポート問題**
   - 外部リソースがアクセス可能か
   - フォントが利用可能か
   - 画像形式が対応しているか
